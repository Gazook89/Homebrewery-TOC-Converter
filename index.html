Legacy ToC to v3 ToC

Recommended usage is for taking the default Legacy Table of Contents and turning it into a format that will work with v3's new styling. Optionally, convert your Legacy ToC to v3 syntax for use with v3 renderer.*

This should work with any default Legacy ToC....any customizations likely won't be carried over.

  1. Create a new Legacy Table of Contents using the snippet from the Snippet Bar on Homebrewery.
  2. Copy the entire generated snippet text, including the opening and closing div tags.
  3. Paste into the "Legacy" text area.
  4. Choose a conversion.
  5. Copy the outputted text into the correct editors of your brew (Text Editor and Style Editor, if needed).
* I don't recommend converting from Legacy to v3 syntax using this...if you are converting your whole brew from legacy to v3, it is likely better to delete the legacy ToC, do your conversion, and then generate a new v3 ToC using the v3 snippet.
